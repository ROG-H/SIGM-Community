CONSULTA EXPEDIENTES
Desde esta pantalla se consultarán los expedientes asociados al usuario conectado.

Para consultar los datos de un expediente concreto, pulsar sobre el expediente.

Para realizar una búsqueda se seleccionará la opción Buscar expedientes de la parte superior de la pantalla.

Se buscarán expedientes introduciendo una fecha concreta (menor que, igual a o mayor que) o un rango de fechas (desde-hasta). El resultado de la búsqueda se muestra en formato lista.

Se seleccionará uno de los expedientes para consultar sus datos. En cada expediente se mostrará la Historia del Expediente. Para la visualización de los diferentes estados del expedeinte se han usado los siguientes iconos:

  •  Estado actual.
  • Estado histórico.
Los documentos asociados a un expediente se muestran a la derecha de cada uno de ellos. Para consultar un documento específico, pulsar sobre el mismo
